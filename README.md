# Whither

HTTP service that sends 302 redirects for Wikipedia entities.

Examples:
- [https://whither.link/Anna's_Archive](https://whither.link/Anna's_Archive) ([Wiki](https://en.wikipedia.org/wiki/Anna's_Archive))
- [https://whither.link/Z-Library](https://whither.link/Z-Library) ([Wiki](https://en.wikipedia.org/wiki/Z-Library))
